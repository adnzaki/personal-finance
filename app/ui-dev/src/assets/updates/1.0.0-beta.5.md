## Update v1.0.0-beta.5🔥🚀

Halo, **SobatUang!** Hari ini (16 Januari 2026) kami merilis **SisaUang** versi **1.0.0-beta.5** yang membawa beberapa peningkatan diantaranya:

- **[Baru]** Mengganti logo dengan logo baru yang lebih modern dan menarik
- **[Peningkatan]** Menambahkan informasi untuk pengguna baru yang belum menambahkan pemilik dan sumber dana
- **[Sistem]** Mengimplementasikan sebagian API untuk fitur laporan

Versi baru ini juga menandai dimulainya era server baru dari _shared hosting_ ke _dedicated server_. Dengan migrasi ini, diharapkan dapat memberikan pengalaman pengguna yang lebih baik dengan adanya peningkatan performa dan kestabilan aplikasi. [Klik di sini untuk melihat perbedaan keduanya](https://copilot.microsoft.com/shares/tmxvQbFDV29EJE2BZkw5k)

### _What's Next?_

Fitur laporan kemungkinan belum dapat dirilis dalam waktu dekat dikarenakan keterbatasan waktu dan sumber daya yang ada.
Sebagian API yang telah dikembangkan belum dapat dilanjutkan karena menunggu proses riset ulang mengingat adanya peralihan dari _planning_ berbayar via **SisaUang Pro** menjadi gratis sepenuhnya.
