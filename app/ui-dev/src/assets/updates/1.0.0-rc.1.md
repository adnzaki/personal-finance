## Update v1.0.0-rc.1🔥🚀

Halo, **SobatUang!** Hari ini (14 Juli 2026) kami merilis **SisaUang** versi **1.0.0-rc.1** yang membawa beberapa perbaikan diantaranya:

- **[Perbaikan]** Memperbaiki bug date filter tidak muncul di ukuran layar medium (1024 - 1439.99px)
- **[Perbaikan]** Memperbaiki ukuran date filter yang tidak konsisten di layar tablet dan smartphone
- **[Perbaikan]** Memperbaiki bug saldo awal dan akhir tidak muncul jika saldo minus
- **[Perbaikan]** Memperbaiki bug tombol kembali tidak berfungsi pada menu dukungan dan informasi
- **[Sistem]** Memperbarui sistem API _backend_ dan antarmuka aplikasi

Versi **rc** merupakan singkatan dari "Release Candidate" yang artinya SisaUang semakin mendekati versi final 1.0.0. Versi **rc** ini dirilis untuk memastikan tidak ada _critical bug_ seperti kesalahan perhitungan, _typo_ maupun antarmuka dan logika aplikasi. Jika dalam versi ini sudah tidak ditemukan bug seperti yang disebutkan, maka versi final siap dirilis ke publik.

### _What's Next?_

Dengan rendahnya minat terhadap fitur **Laporan**, kami memutuskan untuk tidak melanjutkan pengembangan fitur ini dalam waktu dekat. Pembaruan SisaUang ke depannya akan fokus pada pengalaman pengguna dalam menggunakan SisaUang, termasuk kemungkinan dukungan **mode offline**.
